Project Life Cycle

Initiating Phase

At this stage, the study of the project idea is identified and defined, and the tasks and roles required for project management are determined

Planning Phase

Preparing the project and conducting technical, financial, marketing and other necessary studies, which are essential elements for the project feasibility study.

Implementing Phase

Determine the start time of implementation and activities and deliver outputs as planned

Controlling Phase

Evaluate the project's performance from the financial and economic point of view

Closing Phase

At this stage, the project is delivered and the strengths and weaknesses of the project are assessed to be used in future related work
